ImRecruitable (Jun 2018 - Jan 2020)

Description

The ImRecruitable Athletic Recruiting App helps high school athletes with college recruiting. It provides tools to showcase skills, connect with coaches, explore scholarships, and learn about NCAA eligibility.

  • WEBSITE https://app.imrecruitable.com
  • STACK React, Redux, Node.js, Express, MongoDB, Stripe, Heroku
